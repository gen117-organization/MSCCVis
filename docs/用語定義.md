# 用語定義

## クローン種別
- Inter-service clone: 異なるサービス（コードベース）にまたがるクローン。
- Within-service clone: 同一サービス内で完結するクローン。

## コード種別
- Production: パスまたはファイル名に大文字小文字を区別せず `test` を含まないファイル。
- Testing: パスまたはファイル名に大文字小文字を区別せず `test` を含むファイル。
- Mixed: Production と Testing の両方が混在するクローンセット。

## クローンセット
- 同一の clone_id に属するクローン片（fragment）の集合。

## クローン率
- 対象ファイルにおいて、クローンに該当する行の割合。
- コメント・空行・生成コードの除外は行わず、すべて分母に含める。

## コモディフィケーション（共修正）
- 同一クローンセット内で、同一コミットにおいて2片以上が `modified` と判定された状態。
